#+title: Plant Journal Plan
#+author: Matthias

* Vision
Build a local-first plant journal to track plants, care history, and reminders.

* Features
** MVP
- [X] SQLite schema + migration runner
- [X] Plant CRUD (repository + basic UI)
- [X] Journal entry logging (repository + basic UI)
- [X] Journal entry editing (update/delete)
- [X] Watering/fertilizing quick log + type presets
- [X] Batch watering/fertilizing across plants
- [X] Plant schema expanded to match XLSX fields
- [X] Light/Dark theme toggle (Material)
- [X] Reminders (multi-plant, settings, basic UI)
- [X] Initial test suite (Qt Test + Qt Quick Test)
** Later
- [ ] Import from XLSX
- [ ] Notifications / scheduled reminders
- [ ] Analytics / care history summaries

* Decisions
- Store dates in ISO 8601 text fields.
- No DB access from QML; use ViewModels.
- Migrations required for schema changes.
